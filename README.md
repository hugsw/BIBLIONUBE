# Proyecto: Biblioteca"

Este es un proyecto personal de una tienda de libros (biblioteca) desarrollado con HTML, CSS y JavaScript. El sitio incluye un catálogo de productos que se carga dinámicamente, un modal de inicio de sesión y un formulario de registro.
